Projeto de Controle de Estoque com SQL

Objetivo

Desenvolver um sistema de controle de estoque utilizando SQL para analisar produtos, movimentações e identificar necessidades de reposição.

O projeto simula um cenário real de empresas que precisam monitorar entradas, saídas e o valor total do estoque.

---
🛠 Tecnologias utilizadas

- MySQL
- Power BI

---
Análises realizadas

Durante o projeto, foram desenvolvidas consultas para responder perguntas de negócio:

-  Valor total em estoque
-  Produtos com estoque baixo
-  Produtos mais vendidos
-  Produtos menos vendidos
-  Produtos que nunca venderam
-  Diferença entre entradas e saídas
-  Produtos que precisam de reposição

---

📊 Dashboard

O projeto inclui a criação de um dashboard no Power BI com as seguintes métricas:

- Valor total em estoque
- Total de produtos
- Produtos mais vendidos
- Produtos parados
- Controle de entradas e saídas
- Alertas de reposição
